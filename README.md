# Husepoeng — Det magiske observatoriet

Live: https://YOURUSERNAME.github.io/REPONAME/

## About

A magical house-points scoreboard for four houses (Griffing, Håsblås, Ravnklo, Smygard).
Click +1/+5/+10 or -1/-5/-10 to adjust. Gems fall from the top of the hourglass into the
bottom chamber as points accumulate. Data is stored in your browser's localStorage.

## Deploy

1. Put `index.html` in the root of your GitHub repository.
2. Settings → Pages → Deploy from branch → main → / (root) → Save.
3. Wait ~1 minute. Your URL appears at the top of the Pages settings page.

## Notes

- Scores are per-browser. If you open in another browser or clear data, they reset.
- Works 100% offline once loaded.
- Open Tweaks from the toolbar to rename houses, change point values, etc.
